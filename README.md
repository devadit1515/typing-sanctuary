# Multiplayer Speed Typing Game

A real-time multiplayer typing speed game where players compete to type passages the fastest!

## Features
- Create or join game rooms with room codes
- Real-time player progress visualization
- WPM (words per minute) calculation
- Accuracy percentage tracking
- Live countdown before game starts
- Winner announcement with detailed stats

## Installation

1. Install dependencies:
```bash
npm install
```

2. Start the server:
```bash
npm start
```

3. Open your browser to `http://localhost:3000`

4. Share the room code with friends to play together!

## How to Play

1. Enter your name
2. Create a new room or join an existing one with a room code
3. Wait for other players to join
4. When ready, the host can start the game
5. Type the displayed text as fast and accurately as possible
6. See your results and compete with friends!

## Tech Stack
- Frontend: HTML5, CSS3, Vanilla JavaScript
- Backend: Node.js + Express
- Real-time: Socket.io
